## [0.1.0] — 2026-06-14

### Added
- Cliente HTTP tipado con métodos GET, POST, PUT, PATCH, DELETE
- Manejo de errores estandarizado
- Inyección automática de tokens JWT
- Soporte para paginación (params: page, pageSize)
- Soporte para subida de archivos (multipart/form-data)
- Tipado genérico con inferencia de tipos
